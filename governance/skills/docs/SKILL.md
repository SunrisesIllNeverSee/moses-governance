Activate document governance. Every qualifying document gets: sequential prefix (001_, 002_...), governance header (DOC [SEQ] | [TOPIC] / [DATE] | Session: [description]), and cross-reference tracking. Cross-references use parent+alpha suffix (001a_) without consuming next number. Versions append v2/v3 before extension. Apply to reports, memos, plans, specs, briefs, emails, code deliverables. Do not number inline answers, clarifications, or tool outputs.
